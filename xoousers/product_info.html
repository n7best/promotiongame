Description:

Xoo Users is a hassle-free PHP Social Media Sign-up & Login Script that allows you to authenticate and manage users in your website. It is actually under constant improvement, proactive bugfixing and it is the most preferred social media login script. Easily integrate Facebook, Google, Yahoo and Linkedin Sign up and login in your website.

By integrating this script into your existing or new website you can create a private user area where visitors to your website have to be logged in to view page content. You can control access by implementing either Role Based Protection or Logged in Based Protection.

Admin Panel:

  • Search Users.
  • Filter Users.
  • Create Users Manually.
  • Delete/Ban Users.
  • Change User's email, password and personal data
  • Manage Roles
  • Block IP numbers
  • See Creation Date.
  • See Activation Date.
  • See Last IP address.
  • Customize Welcome Email.
  • Customize Password Reset Email.
  • Customize Account Activation Email.
  • Reliable and secure MySQL database backend
  • 4 different acount activation method.
  • 4 different password strenght methods.
  • Send Emails in Bulk.
  • Send Email to one User.

Users Front:

  • Facebook One Click Sign Up / Login.
  • Google One Click Sign Up / Login.
  • Yahoo One Click Sign Up / Login.
  • Linkedin One Click Sign Up / Login.
  • Common Sign up & Login by using username and password.
  • Password Reset Link.

User's Dashboard:

  • Update personal data.
  • Update password.
  • Update email.
  • Upload avatar.
  • Use Facebook's profile picture.
  • Log out.
  • See Last IP Login.
  • See Last Login Date.
